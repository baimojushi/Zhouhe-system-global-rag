# 前端易用性重构说明（2026-07-19）

## 本轮目标

- 提高中文正文、辅助文字、表格和按钮的可读性。
- 降低单屏信息密度，使用留白、卡片和分组表达功能归属。
- 面向非专业中文使用者重写字段、状态、操作和提示文字。
- 保留深度管理能力，但将技术标识、内容校验值、模型名和用量等信息收进“高级信息”。

## 知识库管理页

- 将主流程明确为“第 1 步选择知识库 → 第 2 步选择分类 → 第 3 步管理资料”。
- 使用父工作区与三个内嵌子面板表达知识库、分类、资料和详情之间的归属关系。
- 将“目录与文档、摄取任务、审计记录、潜在关联”改写为“资料与分类、导入进度、操作记录、跨库关联”。
- 重做资料详情、标签、其他分类入口、历史版本、导入任务、操作记录和智能分类建议审核。
- 保留批量移动、分类树拖动、版本切换、关联确认/拒绝、分类建议逐项修改/同意/拒绝与整组撤销能力。
- 将技术标识、内容校验值、自定义 JSON、分类结构版本、任务标识、模型名和 Token 用量默认折叠。

## 其他页面

- 主导航改为任务导向的中文名称与说明。
- 检索页将检索参数改写为用户目标，例如“显示几条结果”和“更看重哪种匹配方式”。
- 长期记忆页更换为非技术示例，并使用“决定、事实、总结”等自然语言。
- 运行状态页用“资料检索服务、资料索引服务、本地回答模型、智能分类助手”解释服务职责，连接地址默认折叠。
- 设置页按“本地服务连接、智能分类助手、夜空背景”分组，不常用设置按需展开。
- 顶栏不再直接展示 Weaviate、BGE-M3、Gemma 等实现名称。

## 设计与可访问性基线

- 正文基准字号 16px，主要辅助文字 13–14px。
- 常用按钮和表单控件高度不低于 44px。
- 使用清楚的标题、说明、状态文本和操作反馈，避免只依赖颜色传达含义。
- 保留键盘焦点样式、表单标签、可访问名称和响应式布局。
- 夜间模式的各面板保持半透明，面板间隙保持透明，以露出实拍夜空背景。
